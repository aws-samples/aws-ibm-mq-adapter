# Build the project

`mvn clean install`  

# Run / Debug

`mvn camel:run`

# Secrets 

* mqAdapterTrustStore: IBM MQ Trust store
* mqAdapterKeyStore: IBM MQ Client keystore
* mqAdapterKeyStorePass: Password to Keystore
* mqAdapterTrustStorePass: Password to Truststore
